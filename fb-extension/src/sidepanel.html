Webhook Setup Guide

1. ไปที่ Facebook Developers > App > Webhooks
2. ตั้ง Callback URL: https://your-domain.com/webhook
3. ใส่ Verify Token ตรงกับด้านบน
4. Subscribe to: messages, messaging_postbacks, feed
5. ใช้ ngrok สำหรับ local dev: ngrok http 3000
